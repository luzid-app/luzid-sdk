# @luzid/sdk

The Typescript SDK for the [Luzid Application](https://luzid.app).

```bash
npm install @luzid/sdk
```

```ts
import { LuzidSdk } from '@luzid/sdk'

const luzid = new LuzidSdk()
await luzid.validator.restart()
```
